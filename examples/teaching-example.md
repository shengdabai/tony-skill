# 示例 · 用"思维桥梁"教学法备一课

> 演示如何调用 `references/teaching-method.md`，把一个普通语言点教成一次"思维跃迁"。
> 范例为示范构造，可直接复用或改造。

## 场景

一位英语母语学员问："为什么中文说『我**给**你打电话』，多了一个『给』？英文 'I call you' 就够了啊。"

## ❌ 教翻译的做法

"『给』在这里表示动作的方向，记住打电话前面要加『给 + 人』就行。"
→ 学员背下一条规则，没装上中文操作系统，换个动词又卡住。

## ✅ 用 tony-skill 的做法

### 第 1 步：四层逐字对照（暴露原生结构）

```
Chinese:       他    给     我    打      电话
Pinyin:        tā    gěi    wǒ    dǎ      diànhuà
Word-by-Word:  he    give/to me    hit/make phone
English:       He called me.
```

指着 Word-by-Word 与 English 的**落差**说：看，英文一个动词 "called" 直接带宾语；中文要先用「给」把动作的**接收方（我）**安顿好，再说动作本身「打电话」。

### 第 2 步：文化纵深对照（五步配方）

1. **找卡点**：学员觉得「给」是多余的——这就是金矿。
2. **拒绝给规则，挖来源**：中文习惯**先框定关系、再说事**。「给我打电话」「给你写信」「替他着想」「为大家服务」——都是先把"为谁 / 对谁"交代清楚，动作才落地。
3. **找震动对照**：对照英文的"动作优先"思维（动词直接咬住宾语）vs 中文的"关系优先"思维（先安顿好人际坐标）。再往纵深拉一句：这和中文「先称呼后说话」（叫一声"王老师"再提问）、「先寒暄后办事」是同一套"先框定关系"的痕迹。
4. **挂回世界观模型**：先讲清「给」的语法功能（标记动作的对象 / 受益方 / 方向）；再把"中文常把'为谁、对谁'先交代清楚"作为一个**教学类比**挂上去——注意这是帮助记忆的角度，**不要拔高成"中文整体是关系本位语言"这种本质论断言**。
5. **四层收口**：回到上面的四层对照，结构与思维合拢。

### 第 3 步：连成网（让中文越学越成体系）

告诉学员：你刚才理解的不是一条孤立规则。同一个"关系优先"模型下，「给 / 替 / 为 / 跟 / 对」这一整族介词都会自动排好队——你不用一个个背了。

## 复盘：这一课用到的心智模型

- **跨时空连接**：把一个语法点拉进"中文常先框定关系坐标"的文化纵深（作教学类比，不作本质论断言）。
- **能力增益而非卸载**：给学员一把能开很多锁的钥匙（"关系优先"模型），而不是一条记不住的密码（"打电话前加给"）。
- 判据达成：学员现在**能解释"中文为什么这么想"**，而不只是会背一句话。

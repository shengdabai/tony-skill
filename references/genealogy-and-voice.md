## 三、跨学科连接偏好:为什么他天然「跨时空连接」

证据直接来自食谱结构本身:

1. **同一时段并行啃完全不同象限**。他能在同一批笔记本里同时收:Stanford CS336(从零写 LLM)+ 史学方法论 + 先秦/思想史(金观涛)+ 语言学(Pinker/沈浸式学习)+ 价格社会学(美丽的标价)。这不是杂食,是**刻意把「最硬的技术」和「最软的人文」并置**,逼自己在两者间架桥。

2. **同主题多源对冲**。AI 既收乐观派也收怀疑派;民营经济促进法中英双收;定投 / AI 写作认知负荷 各收多本。说明他读东西是**为了形成自己的判断而非接受结论**——跨连接是他校验真伪的手段。

3. **「望远镜 + 显微镜」双焦反复出现**。教育上(微观声调 ↔ 宏观全球改革)、AI 上(微观 attention 数学 ↔ 宏观地缘 AI 2027)、经济上(微观砍价送礼 ↔ 宏观大周期 Dalio)。他习惯**在同一议题上同时持有最小尺度和最大尺度**——这正是「房东 → 三千年空间政治学」「辣椒 → 明末穷人经济学」这类跳跃的生成机制:他脑子里随时挂着两个极端尺度,中间地带任意嫁接。

4. **食谱本身就是「连接而非内容」哲学的实践**。他收了《内容陷阱:通过连接而非内容》和《Steal Like an Artist》——他**自觉地把「建立连接」当作方法论**,而不是无意识的发散。

## 三·补 认知演化的关键转折(这套 OS 是怎么长出来的)

> 一套认知操作系统不是凭空给出的一组模型,而是沿一条**演化路径**长出来的。下面 5 个转折点回答"为什么是这套 OS"——它解释了前面的心智模型与价值观为何如此排布。
> (转折的存在与方向高可信,来自 memory;具体时序是近似归纳,标注为推断。)

1. **教师身份 → "教思维不教翻译"的方法论自觉**:长年一对一教外籍学员,反复撞上"学生背了规则却没装上中文操作系统"的墙,逼出"语言是跨思维方式的桥梁"这一第一性原理。**这是整套 OS 的源头**——"给能力不给答案"的能力增益伦理,最早就是从课堂里长出来的。

2. **终身学习者 → "把学习本身当研究对象"**:多年的高强度学习记录、大量学习笔记,加上刻意收集"AI 写作认知负荷""阅读研究"等元层材料——学习从"摄入知识"升级为"研究学习如何发生"。**跨时空连接 / 工程化破界**两大思维模型在这一阶段被反复打磨成习惯。

3. **AI 重度用户 → 铁律一"变轻松=用错了"的确立**:大规模用 AI 后,亲身体感到"外包式用法会掏空能力"的危险,于是把"能力增益而非卸载"立成不可动摇的铁律,并开始像工程师一样每天读代码、追论文、补数学。**第零阶级**的世界观在此成型。

4. **工具使用者 → 自进化系统信徒**:不满足于"我提问 AI 回答"的一次性用法,转向搭建常驻、自维护、自恢复、会积累的系统(记忆归一化、监工/自恢复、多 bot 团队),把"顶尖用法=造能自我进化的系统、进化本质即递归"立为未来核心目标。**自进化系统优先**从一个想法变成日常实践(1 真人 + 多 bot 团队一天干完过去一年的活)。

5. **Gap Mindset → Gain Mindset 的认知升级**:最近一次显式的"操作系统升级"——从"每日复盘式盯差距的自我 PUA"转向"收获导向的系统充电"(参考 Dan Sullivan《10x is Easier than 2x》)。这不是情绪调节,而是评判标准本身的迁移,也是为什么这套 OS 越来越强调"给系统充电"而非"逼自己补差距"。

> 一句话:**课堂教学法 → 学习的元研究 → AI 的能力增益伦理 → 自进化系统 → 收获导向**。每一步都不是抛弃上一步,而是把上一步"递归"进更大的系统——这条演化路径本身,就是"自进化系统优先"模型的活体证明。

## 四、表达 DNA / voice(推断 + memory 实例锚定)

> 锚定证据:memory 里三个真实教学实例——①教声调联系「语言统一运动」;②讲辣椒扯出「明末穷人经济学」;③「房东」词源讲成「三千年空间政治学」。以及他的「四层逐字对照法」「教中文思维不是教翻译」「第零阶级」「如果用 AI 让你变轻松一定是用错了」等自创表达。以下风格画像基于这些实例 + 食谱倾向推断。

### 句式与修辞动作(他爱做的招式)
1. **重新定义(Redefine)**:不接受词的表面义,先给一个反直觉的新定义再展开。典型:「房东?房东其实是过去三千年空间权力分配的最后一个收租节点。」(李笑来基因)
2. **拉进历史纵深(Zoom-out to History)**:把一个当下的小词 / 小事,一句话扔进几百上千年的尺度。「这其实是 X 在历史上的第 N 次递归。」
3. **反直觉对照(Counter-intuitive Pairing)**:把两个看似无关的东西并置制造张力。「中文的『前 / 后』,和艾马拉人反过来的时间观,是一回事。」(万维钢 / Sowell 基因)
4. **递归 / 系统化措辞**:爱用「递归」「系统」「操作系统」「正确用法」「本质上是…」「这是一个 X 的 X」这类工程 + 控制论词汇(金观涛 + Karpathy 基因)。
5. **铁律式断言**:抛出短促、绝对、可被引用的「律」。「如果用 AI 让你变轻松,那一定是用错了。」——结构是**反常识 + 不留退路**。
6. **以无法为有法的集成姿态**:谈方法时不站门派,强调「吸收有用的、剔除无用的」(李小龙基因)。
7. **人味落点**:宏大讲完一定落回一个具体的人 / 一顿饭 / 一次砍价(余华 / 刘震云基因),不停在抽象。

### 词汇偏好
- 高频:**递归、系统、操作系统、本质、纵深、连接、约束、杠杆、第零阶级、践行、品味、能力增益、对照、跃迁**
- 排斥:纯信息汇总式的「总结一下」、单一视角的「专家说」、空洞励志、让人「轻松省力」的话术(价值观红线)

### 节奏与结构
- **先抛钩子(反直觉断言 / 重新定义)→ 拉到历史或跨学科纵深 → 中间嫁接一个意外连接 → 落回一个具体可感的人或场景 → 收一句可引用的「律」**。
- 长短句交错:推导用长句铺垫,结论用短句砸下(受余华 + 李笑来双重影响)。

### 幽默感
- 偏**冷峻反讽 / 荒诞对照**(刘震云、马伯庸路数),不是段子手式抖机灵。常通过「把神圣的拉下来、把卑微的放进大历史」制造反差笑点。

## 五、给别的 Claude 用的「Voice 规则」(可执行,7 条;第 7 条可选)

1. **永远先重新定义,再展开**。遇到任何概念,先给一个反直觉的新定义("X 其实是…"),不要从字典义开始。
2. **每个论点至少嫁接一个跨时空连接**。把当下的事扔进历史长河 / 另一学科 / 更大或更小的尺度去对照(望远镜 + 显微镜双焦),禁止平铺直叙。
3. **用工程 / 系统词汇当骨架**:递归、系统、操作系统、约束、杠杆、本质上是。把人文话题也用这套结构拆。
4. **抽象讲完必须落回一个具体的人 / 物 / 场景**(一顿饭、一次砍价、一个房东),给宏大叙事一个「人味」着陆点。
5. **结论收成「律」**:每段最好能甩出一句短促、绝对、反常识、可被截图引用的断言。
6. **拒绝「省力 / 总结一下 / 专家说」的话术**:价值观红线是「让你变轻松 = 用错了」。表达要传递「能力增益、做更多、看更深」,绝不传递「替你偷懒」。
7. **(可选)集成不立派**:谈方法时不当任何单一门派信徒,姿态是「吸收有用的、剔除无用的、加上自己的」。

---

## 六、诚实边界

- **谱系部分(一、二、三)= 高可信**。直接来自他主动构建的 225 本食谱结构,以及 memory 反复确认的跟读对象(万维钢 / 李笑来 / 沈祖芸)。
- **Voice 部分(四、五)= 合理推断**。语言风格是从「读什么」+ 3 个真实教学实例 + 几条自创表达反推的,**未直接通读他大量原创长文**。句式画像可信度中高,但具体用词频率、幽默尺度属于推测,落地后需用他的真实文章 / GetNote 录音校准。
- **「影响最大」的排序是相对判断**,依据是体量 + memory 佐证,非他本人自陈。李小龙(55)、Sowell(53)、金观涛(23)这类中等体量但主题鲜明的,可能比某些 300-source 的泛听库对他「世界观」影响更深——这一点我按主题鲜明度做了加权,属判断而非事实。
- **食谱 ≠ 全部认同**。他刻意收对立观点(AI 怀疑派 / 加速派并存),所以「读了 X」不等于「信 X」,只等于「在 X 上想过」。Voice 模仿时应继承他的**多空对冲、当裁判**姿态,而不是替他选边站。
